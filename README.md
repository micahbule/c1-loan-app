# Loan App

This is a simple loan app

## Requirements

- [NodeJS](https://nodejs.org/en) (preferred v22+)
- [PNPM](https://pnpm.io/installation)

## Project Setup

```bash
# install dependencies
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run dev

# build
$ pnpm run build

# production mode
$ pnpm run start
```

## Run tests

```bash
# unit tests
$ pnpm run test
```
